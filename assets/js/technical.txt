Update Functions

1. Category Update Functions
updateCategoryDisplay(categoryId, data) - Main function for category updates
updateCategoryTitle(category, newName) - Updates category name
updateCategorySettings(category, data) - Updates width, description, favicon preferences

2. Bookmark Update Functions
updateBookmarkDisplay(bookmarkId, data) - Main function for bookmark updates
updateBookmarkTitle(bookmark, newTitle) - Updates bookmark title and link
updateBookmarkUrl(bookmark, newUrl) - Updates bookmark URL
updateBookmarkDescription(bookmark, newDescription) - Updates description with proper DOM manipulation
updateBookmarkFavicon(bookmark, newFaviconUrl) - Updates favicon with proper URL formatting
updateBookmarkCategory(bookmark, newCategoryId) - Moves bookmark between categories

3. Page Update Functions
updatePageDisplay(pageId, data) - Updates page name and attributes
